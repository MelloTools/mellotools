…and more — every tool is one focused page, no clutter.

## Why it's different

- **Private by design.** Everything runs client-side. Files like PDFs and photos are processed in your browser and never sent to a server.
- **No friction.** No login, no "create an account to download," no upsell wall.
- **Fast & free.** Static pages, instant load, ad-supported so the tools stay free.
